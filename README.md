# EduVision | Project - 6 | Team - 15 (EdgeVision)


**Project Description**
The Face Recognition Attendance System is a project that aims to automate the process of taking student attendance in a classroom setting. The system uses face recognition technology to identify students as they enter the class, eliminating the need for manual attendance taking. The system will update the attendance in real-time into Canvas, a popular Learning Management System (LMS).


**Functional Requirements**
- Real-time Canvas Attendance Sync (Basic)
- Late Arrival Tracking
